Captura de Fotos IA README
==========================

Este es un archivo README para el programa de Captura de Fotos
IA. Si está leyendo esto, probablemente ha ejecutado el index.html
en su sistema.

------------------------------

Si desea usar este programa en otro sistema Windows, transfiera
todos los arhivos contenidos en el directorio "Captura de Fotos
IA" y ejecute el programa Web Server for Chrome en su host Local.

Este programa es de código abierto, por lo que podrá hacer uso del
mismo y modificarlo a corde a sus necesidades.

------------------------------

*Software utilizado*

Para este proyecto se utilizará el lenguaje de marcado HTML en su 
versión 5, dos librerías de código abierto, libres, y gratuitas 
(FLOSS - Free/Libre Open Source Software) y un servidor local:

- html5
- ml5.js
- p5.js
- Bootstrap
- Web Server for Chrome
